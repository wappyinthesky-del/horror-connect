# HorrorConnect 👻

- **名前**: HorrorConnect
- **目標**: ホラー好きのためのWebアプリ

- ✅ モバイルファーストデザイン
- ✅ Noto Sans JP/Noto Sans フォント
- ✅ ゴーストキャラクター（アニメーション削除済み）
- ✅ 添付画像参考のボタンデザイン
